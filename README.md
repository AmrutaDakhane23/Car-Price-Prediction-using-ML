# Car-Price-Prediction-using-ML

This project predicts the selling price of used cars using machine learning.

## Dataset
Car Price dataset containing features like:
- vehicle_age
- km_driven
- fuel_type
- transmission_type
- engine
- max_power
- mileage

## Models Used
- Linear Regression
- Random Forest Regressor

## Result
Random Forest performed better with higher R² score.

## Technologies
Python, Pandas, NumPy, Scikit-Learn, Matplotlib
